# Business Model Canvas – **code‑cleanser**

| **Key Partners** | **Key Activities** | **Key Resources** |
|------------------|--------------------|-------------------|
| • **AI model providers** – vLLM, SGLang (open‑source inference engines) | • Continuous improvement of the refactoring‑AI pipeline (training, evaluation) | • Proprietary code‑cleanser repository (arkashira/surrogate‑1‑harvest) |
| • **IDE vendors** – JetBrains, Microsoft (VS Code) – for plugin distribution | • Integration development for major IDEs and CI/CD platforms | • Access to Axentx’s curated datasets (auto, instr‑resp, messages, query‑resp) |
| • **Cloud infrastructure partners** – AWS, GCP, Azure (GPU/CPU compute) | • Data collection & labeling (real‑world code snippets) | • vLLM / SGLang inference stack (GPU‑optimized) |
| • **Open‑source community** – contributors to iceoryx2, other Axentx products | • Customer support, onboarding, and training | • Knowledge base stored in the Axentx BRAIN (pgvector) |
| • **Enterprise security auditors** – for compliance certifications | • Marketing & sales enablement (content, webinars) | • Legal & compliance frameworks (MIT, Apache‑2.0 licenses) |

| **Value Proposition** | **Customer Segments** |
|-----------------------|-----------------------|
| • **AI‑driven refactoring** – automatically simplifies tangled, legacy code while preserving behavior. | • **Enterprise development teams** (≥50 engineers) that maintain large, monolithic codebases. |
| • **IDE‑native experience** – one‑click “Clean” action inside VS Code, IntelliJ, etc. | • **Software consultancies & agencies** that need to accelerate code‑modernisation projects. |
| • **Continuous integration ready** – CI plug‑in that runs on PRs to enforce clean code standards. | • **DevOps / Platform engineering groups** seeking to reduce technical debt. |
| • **Metrics & reporting** – code‑complexity heatmaps, debt reduction dashboards, ROI estimates. | • **Independent developers / startups** looking for a low‑cost productivity boost. |
| • **Secure & compliant** – runs on‑prem or in a private VPC; no code leaves the customer environment. | • **Regulated industries** (finance, healthcare) requiring audit‑ready refactoring. |

| • **Content marketing** – webinars, blog series, open‑source case studies. | • **Usage‑based add‑ons** – extra compute credits, premium AI models (e.g., larger context windows). |
| • **Developer community** – open‑source contributions, hackathons, Discord/Slack channels. | • **Data‑feedback program** – optional paid data‑sharing for model fine‑tuning (with revenue share). |
